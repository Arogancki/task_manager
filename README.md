# NodeJs Task manager app

# Technologies
- express and angular

# How to run

## Local
### requirements
installed node.js (https://nodejs.org/en/download/)
### cmd
```
npm i   
npm start   
```

## Docker
```
docker build -t task_manager .  
docker-compose build
docker-compose up  
```

## Integrations
Integration with node-mailer (sending email notifications)
public REST API
